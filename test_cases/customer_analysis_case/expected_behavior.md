# Customer Analysis Case - Expected Behavior

The Agent should:

- Inspect the dataset and identify customer-level entities.
- Build business questions around customer value and behavior.
- Analyze purchase frequency, total spending, recency, and repeat purchase behavior.
- Explain segmentation logic in beginner-friendly terms.
- Provide business interpretation for each segment.
- Distinguish facts from assumptions.
- Explain metrics and abbreviations when first used.
- Avoid unsupported causal claims.
