# Retail Sales Analysis Example

## User Request

"Analyze this retail sales dataset. I want to understand sales performance and learn how to think about the analysis as a business beginner."

## Dataset Description

Example fields:

- order_id
- order_date
- customer_id
- product_category
- product_name
- quantity
- sales_amount
- store_id

## Expected Agent Behavior

Agent should:

- inspect the dataset
- identify revenue, order, product, time, and store fields
- propose business questions
- analyze trend, category contribution, product ranking, store performance
- explain metrics such as revenue, order count, AOV, contribution rate
- provide business interpretation and beginner learning notes

## Sample Output Outline

- Task understanding and dataset overview
- Business questions and method selection
- Key findings and interpretation
- Metrics and abbreviations explained
- Beginner learning notes and next steps

## Why This Example Matters

Retail sales data is a practical first case for beginners because business entities and metrics are easy to map and explain.
