# Customer Segmentation Example

## User Request

"Please analyze this customer dataset and help me understand customer segmentation as a beginner."

## Dataset Description

Example fields:

- customer_id
- order_date
- order_id
- sales_amount
- product_category
- region

## Expected Agent Behavior

Agent should:

- inspect customer-level fields
- explain purchase frequency, total spending, and recency
- segment customers into interpretable groups
- explain segment logic in plain language
- connect segment results to practical actions

## Sample Output Outline

- Task and segmentation goal
- Dataset overview
- Segmentation method and rationale
- Segment-level findings
- Beginner learning notes

## Why This Example Matters

It teaches beginners how customer behavior fields become actionable business segments.
