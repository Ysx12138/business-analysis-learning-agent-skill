## 2. Self-Check Questions (5)

01. What total metric can be decomposed, and by which dimensions?
02. After decomposition, how large are the subgroup behavior differences (including multiples)?
03. What missing key information can be inferred by a proxy variable?
04. Is the observed difference a preference or a constraint?
05. Which segment is both high-share and high-difference (a leverage point)?

## 3. PDF Output Checklist

When generating the PDF thinking model report, verify:

- [ ] Cover page (dataset name, row count, key metrics, analysis mode)
- [ ] Executive summary (1 paragraph + 5-6 key metrics)
- [ ] Each model (A-E): data evidence, business insight, beginner takeaway
- [ ] Insight box or highlight for the most important finding per model
- [ ] Self-check questions with complete answers
- [ ] Action recommendations with measurable targets
- [ ] Page numbers and consistent formatting
