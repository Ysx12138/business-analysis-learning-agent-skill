# Usage Guide

## 1. What This Skill Is For

This skill supports learning-oriented business data analysis for beginners.

## 2. Recommended Use Cases

- retail sales analysis
- customer segmentation
- product performance analysis
- marketing campaign analysis
- store operation analysis

## 3. How to Use with an Agent

Example prompt:

"Use the Business Analysis Learning Agent Skill to analyze this dataset. Please explain the business logic, metrics, and beginner learning points while analyzing."

## 4. Recommended Dataset Types

- CSV or Excel business datasets
- transaction records
- customer behavior data
- product performance datasets
- campaign performance datasets

## 5. Output Structure

Recommended sections:

1. Task Understanding
2. Dataset Overview
3. Business Questions
4. Analysis Methods
5. Key Findings
6. Business Interpretation
7. Metrics and Concepts Explained
8. Recommendations
9. Beginner Learning Notes
10. Limitations and Next Steps

## 6. Tips for Better Results

Provide:

- dataset
- business context
- target question
- beginner learning goal
- preferred output format

## 7. Thinking Models (Learning Goal)

To keep the analysis transferable for beginners, require the Agent to teach at least one reusable thinking model:

- Decomposition (structure breakdown)
- Segment behavior divergence (layered differences)
- Proxy inference (indirect inference)
- Constraint vs preference (constraint as explanation)
- Leverage point identification

Reference template: `templates/thinking_models_template.md`.
