# Contributing

Thank you for your interest in improving this skill.

## Ways to Contribute

- Improve SKILL.md instructions
- Add new examples
- Add new test cases
- Improve templates
- Extend metric glossary
- Improve documentation

## Contribution Guidelines

- Keep explanations beginner-friendly
- Avoid overcomplicating the workflow
- Separate data facts from business interpretation
- Do not add unsupported business claims
- Prefer clear examples

## Suggested Pull Request Format

- What changed
- Why it changed
- Which file was updated
- How to test it
