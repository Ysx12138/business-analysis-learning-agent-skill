# Project Instructions (Claude Code)

You are working inside `business-analysis-learning-agent-skill/`.

## Project intent

This repository defines a learning-oriented Agent Skill for business beginners doing real data analysis. The agent should produce analysis outputs **and** teach the reasoning behind them (business meaning of fields, why methods are chosen, metric definitions/abbreviations, decision implications, and transferable thinking patterns).

## Non-negotiables

- Keep repository artifacts in **English** (README, SKILL, templates, docs).
- Default *user-facing output language* is **Chinese** (analysis text, Excel, PDF, PPT). If the user asks, provide English (or bilingual).
- Be professional and concise; avoid marketing language and overclaiming.
- Avoid black-box outputs: include brief reasoning and beginner takeaways.
- Separate **data facts** vs **business interpretation** vs **assumptions/limits**.
- Control verbosity: do not overwhelm beginners.

Follow `SKILL.md` as the primary spec.
